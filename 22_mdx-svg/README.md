# 22_mdx-svg
[../22_mdx-svg](../22_mdx-svg)

Example for testing SVG usage within `.astro`, `.md` and `.mdx`

critical features under test :
* usage of svg `<defs/>` tag
* usage of attributes with ':' e.g. `xmlns:xlink`
* usage of html comment inside svg
